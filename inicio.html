HTML

Es el lenguaje de marcado estándar utilizado para crear y estructurar páginas web. Es la base de cualquier documento web y define la estructura del contenido, utilizando una serie de elementos y etiquetas.

CSS

Es un lenguaje utilizado para describir la presentación visual de un documento HTML. CSS se define cómo se ve ese contenido, incluyendo aspectos como colores, fuentes, disposición de los elementos, espaciado, y más.

JavaScript

Es un lenguaje de programación de alto nivel y orientado a objetos que se utiliza principalmente para desarrollar sitios web dinámicos e interactivos. Es uno de los pilares fundamentales del desarrollo web junto con HTML y CSS.
